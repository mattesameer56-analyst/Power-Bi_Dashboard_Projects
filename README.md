## 🛒 Project 2: Supermarket Sales Dashboard

### 📝 Project Overview
An interactive retail sales dashboard designed to analyze supermarket transaction data, track seasonal trends, and breakdown revenue across key product lines and customer segments.

### 🔑 Key Insights & Features
* *Interactive Navigation:* Built customized buttons (Customer Type, Payment Method, Gender, etc.) for dynamic chart updates.
* *Sales Performance:* Visualized month-on-month sales comparisons for 2019 and 2020 to track peak periods.
* *Granular Data View:* Integrated structured tables showing transaction details, tax calculations, and gross income metrics.
* *Tech Stack:* Power BI Desktop, Power Query, Custom Page Navigation.

---

## 🛠️ Skills Demonstrated
* *Data Transformation:* Power Query, Data Cleansing, M-Code.
* *Data Modeling:* Creating Relationships (Star Schema), Calculated Columns, and Measures using DAX.
* *Data Visualization:* KPI Cards, Interactive Filters/Slicers, Bar/Line charts, and Matrix tables for deep dives.
