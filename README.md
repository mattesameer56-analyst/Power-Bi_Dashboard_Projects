### 📝 Project Overview
This dashboard helps HR managers and leadership track key employee metrics, monitor retention, and understand attrition patterns across different departments and demographics.

### 🔑 Key Insights & Features
* *Attrition Analysis:* Monitored total attrition rate based on age groups, gender, and job roles.
* *Employee Demographics:* Visualized workforce distribution by department (Sales, R&D, HR) and education fields.
* *Job Satisfaction:* Mapped performance ratings against work-life balance to identify teams with high burnout risk.
* *Tech Stack:* Power BI Desktop, Power Query for ETL (Data Cleaning), DAX for advanced measures.
